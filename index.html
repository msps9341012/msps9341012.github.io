    Expert Trading Strategy Imitator

    Python | Deep Learning | Tensorflow

    Although financial market is complex and hard to surrive, there are still some experts having unique decision rules that help them to gain more profit than othets. In this work, we would like to imitate those experts' trading behavior and clone their professional strategies. Our framework is based on

    • Trading analysis: To discover experts from real transaction data
    • Generative Adversarial Imitation Learning (GAIL)

    • Data augmentation technique: To deal with unseen states or distribution mismatchm, inspired by DAGGER

    For result,

    • In cloning expert part (i.e. buying and selling at similar time), our framework reached 85.41% accuracy on testing data.
    • In backtesting part, our cloned-strategy otained the best average ROI and the lowest risk compared with other tradtional strategies.

    Investor Diagnostics System

    Python | Javascript| Machine Learning

    In this work, we designed an Investor Diagnostics System which can define an investor's type (high risk/ moderate..etc) and offer customized investment advice. The project can be divided into three parts.

    • Clustering Model
      • Analyzed millions of investors' investment behavior and extracted trading features from it. (ex: trading preference, Value at risk, Profit)
      • Filter outlier using Isolation Forest and clustered the remaining with K-Means and trading features into 10 groups.
    • Personal Suggestions
      • To obtain characteristics of a given group, applied Decision tree to compare it with the rest, i.e. the difference between this group with others. And utilized those characteristics to decide which trading type of a group.
      • For each group, display the top 20% high-profit investors' trading features as suggestions.
    • Web visualization
      • Employed javascript and D3.js to visualize each investor's trading features.
      • Provided search page that user can find their suitable financial instruments with given criteria.

    PicDia

    Android | PHP | MySQL |MatLab

    PicDia is a mobile application that provides children a new way to learn English. The whole concept is, "What you see is what you learn!". Whenever children see an item that they do not know how to say it in English, they can just simply take a picture of it using PicDia. PicDia will recognize it, pronounce its name and show some information about this word. Learning by discovering in the real world, we hope our app can help children learn English in a fun and efficient way.
    PicDia offere the following functions,

    • Object recognition: Once user takes a picture, the photo will be uploaded to a PHP server and processed by VGG16 running through Matlab. When finished, the server would send the classification result back to the user.
    • Pronounce Word: Read the searched word through TexttoSpeech Api.
    • Dictionary: using Jsoup to crawl information on Yahoo Dictionary
    • Enhance Pronunciation: Employed SpeechRecognizer Api to make sure children pronounce correctly.
    • Review and Quiz: Utilized SQLite to record what children have learned and provide some quiz to help them remember these words.
